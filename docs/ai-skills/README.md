# Ask AI Skills

These docs are reusable analysis playbooks for TARS Ask AI routes.

Each skill should be written so it can be used as prompt context, QA criteria, or implementation guidance. Keep skills practical and focused on decisions the app needs to make.

## Current Skills

- [Comprehensive company analysis](./comprehensive-company-analysis.md) - use for stock-detail "Analyze company with AI" and broad company due diligence.
- [13D/13D/A filing intent](./13d-13da-filing-intent.md) - classify investor intent from Schedule 13D and 13D/A filings.

## Skill Format

Use this structure for new skills:

1. Overview
2. When to use
3. Source sections or data fields to prioritize
4. Classification rules
5. Red flags and nuance
6. Output template
